## An election twitterbot in Python and R

By [Wes Stephenson](https://twitter.com/wesstephenson) and [Alison Benjamin](https://twitter.com/hey_benjamin).

### Setup

Requires Python 3

```
python3 -m venv venv
source venv/bin/activate
pip3 install -r requirements.txt
make run_twitterbot
```

### Run 

```
make run_twitterbot
```
